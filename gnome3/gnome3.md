# GNOME 3
## Autostart
```
# ~/.config/autostart/foobar-baz.desktop

[Desktop Entry]
Name=MyScript
GenericName=A descriptive name
Comment=Some description about your script
Exec=/path/to/my/script.sh
Terminal=false
Type=Application
X-GNOME-Autostart-enabled=true
```
